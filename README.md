# Node-RED Lecture

To install addtional nodes, run the following command in your Node-RED user directory `./local`. This directory is created when you first run Node-RED.

```powershell
pnpm run start
```

- Wait for Node-RED to start and then stop it (Ctrl + C)

```powershell
cd ./local && pnpm install @sumit_shinde_84/node-red-dashboard-2-ui-webcam && cd ..
```
